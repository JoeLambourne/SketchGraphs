sketchgraphs.pipeline.make_sketch_dataset¶

This script is responsible for creating the main sketch dataset from the JSON files.

Some basic filtering is applied in order to exclude empty sketches. However, as this dataset is intended to capture the original data, further filtering is left to scripts such as sketchgraphs.pipeline.make_sequence_dataset which process dataset for learning.
