sketchgraphs.pipeline.graph_model.quantization.QuantizationMap¶

class sketchgraphs.pipeline.graph_model.quantization.QuantizationMap(values=None)¶

Utility class representing a quantization map.

This mapping can be used by accessing the call and index operator of an instance (similarly to how an Enum behaves).

If qmap is a quantization map, qmap[value] returns the index for the given value, and qmap(index) returns the value for the given index.
